# 房貸計算器
### 操作說明  
1. 原有設定預設數值
2. 依次輸入：房屋總價、自備款(%或金額)、年利率(%)、貸款年限(年)、寬限期(年)
3. 寬限年可選填
4. 按下「計算」或鍵盤「Enter」，產出數出結果
5. 按下「清除」或鍵盤「Esc」，清除輸入欄位內的所有資料

### 鍵盤操作
1. 於輸入欄位內，按鍵盤「上下鍵」可以跳轉至該欄位之前後欄位，與滑鼠操作不同，會直接全選原資料方便修改
2. 於輸入欄位內，按鍵盤「左右鍵」可以選擇「自備款(%)」或「自備款(金額)」進行計算，並會刪除原本欄位內的資料

### 錯誤提示
自動聚焦到錯誤欄位
1. 空白輸入
2. 格式錯誤(不會讀取非數字的輸入)
3. 超出允許範圍
4. 邏輯錯誤
